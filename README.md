# Pac-Man

Projet de jeu réalisé en Java avec JavaFX, mettant l’accent sur l’architecture logicielle et l’utilisation de patrons de conception.

## Objectif

Développer un clone du jeu Pac-Man afin de mettre en pratique la programmation orientée objet et plusieurs design patterns dans un projet complet et structuré.

## Fonctionnalités

- Jeu Pac-Man jouable avec plusieurs niveaux
- Génération dynamique du labyrinthe
- Comportements variés des fantômes (poursuite, fuite, aléatoire)
- Système de bonus (vitesse, invincibilité, multiplicateur de score, etc.)
- Gestion des états de Pac-Man et des fantômes
- Sprites animés avec JavaFX
- Architecture modulaire et extensible

## Technologies utilisées

- Java
- JavaFX
- Gradle
- Git / GitHub
- Eclipse
- IntelliJ IDEA

## Compétences développées

- Mise en place d’une architecture proche du MVC
- Utilisation des design patterns : State, Strategy, Singleton, Factory, Decorator
- Gestion d’un moteur de jeu en temps réel
- Génération procédurale de carte
- Organisation d’un projet Java modulaire
- Manipulation des propriétés JavaFX et animations

## Ce que j’ai réalisé dans le projet

- Système de génération de carte
- Placement des pac-gommes
- Implémentation du Decorator pour la génération du labyrinthe
- Stratégie personnalisée de génération de carte
- Intégration des méga-gommes
- Gestion des niveaux avec une fabrique abstraite
